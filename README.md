# Pong 双人/人机对战游戏
## 项目简介
本项目基于 **SFML 3.1.0** 图形库开发经典 Pong 乒乓对战小游戏，支持**玩家 VS AI 人机模式**，遵循现代 C++17 编码规范，全程采用 RAII 资源管理原则完成资源生命周期管控。

## 核心游戏功能
1. **操控规则**
   - 玩家：键盘 ← / → 方向键控制屏幕底部横向挡板左右移动；
   - AI：程序逻辑自动运算，控制屏幕顶部挡板自动追踪小球进行防守。
2. **碰撞与得分机制**
   - 小球触碰屏幕上、下边界正常反弹；
   - 小球触碰屏幕左、右侧边判定得分：成功被挡板接住则反弹，未接住对应对手获得分数。
3. **回合重置机制**
   产生得分后，游戏自动暂停 2 秒，自动重置挡板位置与小球初始状态，开启新一轮对局。
4. **UI 显示**
   实时在窗口界面展示玩家与 AI 的双方当前分数。

## 开发环境
- IDE：Visual Studio 2019
- C++ 标准：C++17
- 第三方图形库：SFML 3.1.0（Simple and Fast Multimedia Library）

## 编译配置要点
1. 项目属性 C++ 语言标准设置为 **C++17**；
2. 运行库（MDd/MTd / MD/MT）、编译架构(x86/x64)与所使用的 SFML 3.1.0 编译版本保持完全一致；
3. 头文件、库目录正确配置 SFML 3.1 路径，链接器附加依赖项添加对应 sfml-graphics、sfml-window、sfml-system 模块 lib 文件；
4. 运行时将 SFML 对应 dll 文件放置程序输出目录，字体等资源配置正确路径。

## 编码规范
项目严格遵循 **RAII 资源管理原则**，纹理、字体、窗口、精灵等 SFML 资源随对象生命周期自动创建与释放，避免内存泄漏与野指针问题。
